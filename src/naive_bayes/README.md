# 朴素贝叶斯分类器 (Naive Bayes Classifier)

## 学习目录

本项目将带你一步步实现朴素贝叶斯分类器，帮助你理解其核心原理。

### 文件说明

| 文件 | 内容 |
|------|------|
| `01_theory.py` | 贝叶斯定理基础，通俗解释 |
| `02_gaussian_nb.py` | 高斯朴素贝叶斯（处理连续特征） |
| `03_multinomial_nb.py` | 多项式朴素贝叶斯（处理文本分类） |
| `04_text_classification.py` | 实战：垃圾邮件分类 |

### 核心公式

**贝叶斯定理**:
```
P(类别|特征) = P(特征|类别) × P(类别) / P(特征)
```

- **P(类别|特征)**: 后验概率，我们要求的
- **P(特征|类别)**: 似然，从训练数据统计
- **P(类别)**: 先验概率，类别出现的频率
- **P(特征)**: 证据，可忽略（因为对所有类别相同）

### 为什么叫"朴素"？

因为它假设所有特征之间**相互独立**，这个假设虽然朴素（天真），但在实践中效果出奇的好！
